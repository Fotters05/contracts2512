## 1. ✅ Парсер ДОП - ИСПРАВЛЕНО

**Решение:** Обновлен slug с `'дополнительное-образование'` на `'курсы-для-детей-и-подростков'`

**Файл:** `parser_nodejs/config.js`

## 2. ✅ Автообновление - ИСПРАВЛЕНО

**Проблема:** Приложение не получало обновления после `git push`

**Причина:** GitHub Actions запускается **ТОЛЬКО на теги**, а не на обычные push

**Решение:** 
- Создан скрипт `release.ps1` для автоматизации релизов
- Создана инструкция `КАК_ВЫПУСТИТЬ_ОБНОВЛЕНИЕ.md`
- Создан документ `ПРОБЛЕМА_АВТООБНОВЛЕНИЯ.md` с объяснением

**Как использовать:**
```powershell
# Создать новый релиз
.\release.ps1 -Version 1.1.5
```
