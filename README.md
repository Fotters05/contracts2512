# 📋 Contract2512 - Система управления договорами

WPF приложение для управления образовательными договорами и программами обучения.

## ✨ Возможности

- 📝 Управление договорами и слушателями
- 📚 Каталог образовательных программ
- 🔄 Автоматический парсинг программ с сайта 25-12.ru
- 📊 Генерация документов (Word, Excel, PDF)
- 🔄 Автоматическое обновление через GitHub
- 💾 PostgreSQL база данных

## 🚀 Быстрый старт

### 1. Настройка базы данных

```bash
# Скопируй шаблон настроек
copy .env.example .env

# Запусти приложение - откроется окно настроек БД
```

### 2. Настройка автообновления

Добавь в `.env`:
```env
GITHUB_OWNER=твой-username
GITHUB_REPO=contracts2512
GITHUB_TOKEN=твой-токен
```

📖 Подробнее: [SETUP.md](SETUP.md)

## 📦 Требования

- .NET 8.0
- PostgreSQL 12+
- Node.js 18+ (для парсера)

## 🔧 Разработка

### Сборка проекта

```bash
dotnet build
```

### Запуск парсера

```bash
cd parser_nodejs
npm install
node index.js
```

### Выпуск новой версии

```bash
# Обнови версию в Contract2512.csproj
git tag v1.0.1
git push origin v1.0.1
```

GitHub Actions автоматически соберет и опубликует релиз!

📖 Подробнее: [AUTO_UPDATE_GUIDE.md](AUTO_UPDATE_GUIDE.md)

## 📁 Структура проекта

```
Contract2512/
├── Models/              # Модели данных
├── Services/            # Бизнес-логика
├── Views/               # UI окна
├── parser_nodejs/       # Node.js парсер
├── .env                 # Настройки (не в git)
└── .env.example         # Шаблон настроек
```

## 🔒 Безопасность

- `.env` файл в `.gitignore` - не попадет в репозиторий
- GitHub токен используется только локально
- Пароли БД хранятся только в `.env`
