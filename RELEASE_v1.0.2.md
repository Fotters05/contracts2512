# 🚀 Release v1.0.2

## ✅ Что изменилось:

### Улучшения UI:
- 🔄 Изменена кнопка в окне тех поддержки: "Открыть" → "Связаться"
- 🎨 Добавлен красивый splash screen для установщика
- 📦 Улучшен внешний вид установщика

### Технические изменения:
- Обновлена версия: 1.0.1 → 1.0.2
- Настроен Clowd.Squirrel для отображения splash screen
- Добавлена иконка приложения в установщик

## 📦 Как выпустить релиз:

```bash
# 1. Закоммитить изменения
git add .
git commit -m "Release v1.0.2: Улучшен UI тех поддержки и установщика"
git push origin main

# 2. Создать тег
git tag v1.0.2
git push origin v1.0.2
```

## 🎯 Что произойдет:

1. GitHub Actions автоматически:
   - Соберет проект
   - Создаст Setup.exe с splash screen
   - Опубликует Release на GitHub

2. Пользователи с версией 1.0.1:
   - Увидят окно обновления при запуске приложения
   - Смогут обновиться до 1.0.2 одним кликом

3. Новые пользователи:
   - Скачают Setup.exe
   - Увидят красивый splash screen при установке
   - Получат приложение версии 1.0.2
