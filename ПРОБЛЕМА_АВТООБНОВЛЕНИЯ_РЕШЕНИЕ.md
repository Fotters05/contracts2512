# Решение проблемы автообновления

## Проблема
Приложение не видело обновления, хотя релизы на GitHub создавались успешно.

## Причина
Для **приватных репозиториев** GitHub не позволяет скачивать файлы релизов без авторизации. Squirrel пытался получить доступ к файлу `RELEASES` по URL:
```
https://github.com/Fotters05/contracts2512/releases/latest/download/RELEASES
```

Но получал ошибку 404, так как для приватного репозитория требуется токен доступа.

## Решение
Используем GitHub Personal Access Token для доступа к релизам приватного репозитория.

### Формат URL с токеном:
```
https://TOKEN@github.com/owner/repo/releases/latest/download
```

### Изменения в коде (App.xaml.cs):
```csharp
var githubToken = EnvConfigService.Get("GITHUB_TOKEN");

if (!string.IsNullOrEmpty(githubToken))
{
    // Для приватного репозитория используем токен
    updateUrl = $"https://{githubToken}@github.com/{githubOwner}/{githubRepo}/releases/latest/download";
}
else
{
    // Для публичного репозитория токен не нужен
    updateUrl = $"https://github.com/{githubOwner}/{githubRepo}/releases/latest/download";
}
```

### Настройка токена в .env:
```env
GITHUB_OWNER=Fotters05
GITHUB_REPO=contracts2512
GITHUB_TOKEN=ghp_ваш_токен_здесь
```

## Как получить GitHub Token:
1. Перейдите на https://github.com/settings/tokens
2. Нажмите "Generate new token" → "Generate new token (classic)"
3. Выберите scope: `repo` (Full control of private repositories)
4. Скопируйте токен и добавьте в файл `.env`

## Проверка работы:
1. Убедитесь что в `.env` указан правильный токен
2. Запустите приложение версии 1.3.2
3. Приложение должно обнаружить обновление до версии 1.3.3
4. После установки обновления приложение перезапустится с новой версией

## Важно:
- Токен должен быть включен в установщик (файл `.env` копируется при сборке)
- Для публичных репозиториев токен не требуется
- Токен дает доступ только к релизам, не к исходному коду

## Версии:
- v1.3.2 и ниже - не работало автообновление для приватных репозиториев
- v1.3.3 и выше - исправлено, используется токен из .env
